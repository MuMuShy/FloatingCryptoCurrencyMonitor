# floating-monitor with cryptocurrency
# 浮動視窗虛擬貨幣價格監控
a floating monitor window to show price of cryptocurrency.
use binance api to get price
半透明的浮動視窗讓你方便檢測加密貨幣的價格

### main.py
	python main.py.
### exe file 
    /dist/main.exe
